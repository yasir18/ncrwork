int x[8]= {10,20,30,40,50,60,70,80};


a. What is the meaning of x?
Ans: x refers to the starting/base address of the array


b. What is the meaning of (x+2)?
Ans: (x+2) means it refers to address of third element in the array i.e., address of  30


c. What is the value of *x?
Ans: *x  retrieves the value stored at x location i.e., first element in the array


d. What is the value of (*x+2)?
Ans: (*x+2)  adds an integer value 2 to the value stored at base address i.e., 10 . Thus result will be 12 .

e. What is the value of *(x+2)?
Ans: *(x+2)  retrieves the value stored at (x+2) locations i.e., at third location and prints value 30
